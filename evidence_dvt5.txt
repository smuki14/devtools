# Git log (коммит очистки виден)
git log --oneline -n 3
[fc9a080 (HEAD -> feature/DVT-4, origin/feature/DVT-4) docs: update README.md
 f0114be Очистить репозиторий; обновить .gitignore
 1028839 fix: remove gradle config from MenteeProgress.java; update build.gradle]
Правило качества: git status clean перед push

Перед каждым git push обязательно выполняй:
  git status

Ожидаемый результат:
  nothing to commit, working tree clean

Где проверять:
  - В IDE (Commit Tool Window, Ctrl+K): списки Unversioned Files и Changes должны быть пустыми.
  - В терминале: вывод git status должен быть чистым.
